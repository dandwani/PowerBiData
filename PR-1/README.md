Data Leverager – Power Query Transformation Project

Objective

The objective of this project is to perform ETL (Extract, Transform, Load) operations using Power BI Power Query Editor.
This project focuses on data cleaning, transformation, merging, appending, grouping, pivoting, and data profiling techniques.
